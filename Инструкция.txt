Для запуска необходимо:
0. подгрузить зависимости: npm install
1. Запустить сервер MongoDB: mongosh # (в зависимости от того, куда установлен сервер: C:\Program Files\MongoDB\Server\7.0\bin)
2. Запустить приложение: npm start или node .\server.js

ВАЖНО!
Первый зарегистрированный пользователь получает роль администратора. Далее распределение ролей осуществляется через созданную учетную запись.
